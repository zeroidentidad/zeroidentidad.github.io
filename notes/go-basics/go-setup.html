Configuración entorno de desarrollo

Configurar un entorno de desarrollo de Go implica varios pasos, como instalar el SDK de Go, configurar espacio de trabajo y verificar la instalación. Aquí una guía junto con una verificación de ejemplo:

Paso 1: Descargar e instalar Go

  1. Visitar el sitio web oficial de Go en https://golang.org/dl para descargar la última versión de Go para el sistema operativo de trabajo (Windows, macOS o Linux).

  2. Usar el instalador descargado siguiendo las instrucciones de instalación específicas para cada sistema operativo.

Paso 2: Configurar espacio de trabajo

  1. Crear un directorio que sirva como espacio de trabajo de Go. Por convención, este directorio suele denominarse go y se encuentra en el directorio personal de su usuario. En Windows, puede crear una carpeta go en el directorio de su perfil de usuario, normalmente C:\Users\<Username>\.

  2. Establecer la variable de entorno GOPATH para que apunte al espacio de trabajo de Go.

  3. Agregar el directorio GOPATH/bin a la variable de entorno PATH del sistema para poder ejecutar los binarios de Go instalados por paquetes. En Windows, se puede configurar la variable de entorno PATH de forma similar a la variable GOPATH.

  4. Recargar su perfil de shell o reiniciar terminal para cargar los cambios aplicados.

Paso 3: Verificar instalación

  1. Abrir una terminal o símbolo del sistema.

  2. Para verificar que Go esta correctamente instalado y configurado, ingresar el comando:

go version

Se debería ver la versión de Go instalada, lo que indica que la instalación fue exitosa.

Paso 4: Crear un programa en Go

Crear un archivo de código fuente de Go con nombre main.go utilizando el editor de código preferido.

package main

import "fmt"

func main() {

fmt.Println("Hello, world!")

}

Guardar el archivo con los cambios y regresar a la terminal.

Construir y ejecutar el programa:

go run main.go

La salida será Hello, world! impreso en la terminal, indicando que el entorno de Go está configurado correctamente.
